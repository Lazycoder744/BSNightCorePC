# BSNightCorePC

A Beat Saber PC mod that **removes pitch compensation globally** so songs go *NIGHTCORE*...

## Installation

1. Place `BSNightCore.dll` into your Beat Saber `Plugins/` folder.  
2. Launch Beat Saber.

## Compatibility

- Tested and verified working on 1.40.8, not sure about other versions.

## Usage

Use the UI toggle in the Solo Mod's tab to disable and enable the mod.

## Contributing

Please contribute because I'm new to this and I need help ironing out the bugs!

### Known issues
- Pitch compensation stays disabled after the toggle is disabled. Requires the user to click the speed modifier to OFF in order to remove the patch.
